SOVA Build + Buff Pool v3 (No Stacks)

Buff values are fixed. Floor progression changes only which buffs are unlocked and how often they appear.

Offer rule: each skill screen shows 3 buff options.
Roll rule: chance = buffWeight / sum(unlockedWeights)
Stack rule: if a buff is selected once in the run, it is removed from the offer pool.
